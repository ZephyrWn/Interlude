## Interlude v0.8.0

**Language / 语言：** [中文](#中文说明) | [English](#english)

### 中文说明

Interlude v0.8.0 是 Interlude 的首个公开版本。Interlude 是一个轻量级 Windows 后台工具，用于协调音乐播放器和其他应用的声音播放：当其他应用开始播放声音时，它可以自动暂停你选择的音乐播放器，并且只恢复由 Interlude 自己暂停过的音乐。

#### 主要功能

- 当其他应用播放声音时，自动暂停并恢复已选择的音乐播放器。
- 支持声音活动、媒体播放状态和混合检测模式。
- 可从 Windows 媒体会话中选择默认播放器。
- 支持忽略指定应用，避免这些应用触发自动暂停。
- 支持系统托盘运行、随 Windows 启动、最小化到托盘和关闭到托盘。
- 支持手动恢复音乐、手动播放变更保护、可配置检测时间和渐入渐出音量控制。
- 内置英文和简体中文界面。
- 可快速打开日志目录。

#### 下载说明

- 适用于 Windows 10 / 11 x64。
- 推荐下载 `Interlude-v0.8.0-win-x64.zip`，完整解压后运行其中的 `Interlude.exe`。
- ZIP 中的可执行文件是自包含版本，不需要额外安装 .NET Runtime。
- 如果只想直接运行单个文件，也可以下载 `Interlude-v0.8.0-win-x64.exe`。
- 可使用 `SHA256SUMS.txt` 校验下载文件。

### English

Interlude v0.8.0 is the first public release of Interlude, a lightweight Windows background utility for coordinating music playback with other audio activity.

#### Main Features

- Automatically pauses and resumes the selected music player when other apps produce audio.
- Supports sound activity, media playback, and hybrid detection modes.
- Lets users choose the default player from Windows media sessions.
- Supports ignored apps that should not trigger automatic pause.
- Supports tray operation, start with Windows, minimize to tray, and close to tray.
- Supports manual music restore, manual playback override handling, configurable detection timing, and fade-based volume control.
- Includes English and Simplified Chinese UI.
- Provides quick access to the log directory.
